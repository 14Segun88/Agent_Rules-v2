# Jules Supervisor Task — AGENTS_RULES v2

> **Роль:** Ты — супервизор базы знаний. Твоя задача — проверить здоровье репозитория AGENTS_RULES-v2 и исправить найденные проблемы.

---

## Что делать при каждом визите

### Шаг 1: Запусти механическую проверку

```bash
cd /home/segun/CascadeProjects/Перед\ 0_2/AGENTS_RULES-v2
python3 supervisor.py
```

Прочитай отчёт. В нём будут проблемы с уровнями: 🔴 error, 🟡 warn, 🔵 info.

### Шаг 2: Автоисправь безопасные проблемы

```bash
python3 supervisor.py --fix
```

Это автоматически:
- Удалит ложные ошибки («Всё по плану» из wiki/errors/)
- Удалит дубликаты записей в wiki

### Шаг 3: Интеллектуальная проверка (делаешь ТЫ)

Механическая проверка не видит всего. Проверь вручную:

1. **Прочитай `shared_status.md`** — актуален ли статус? Не устарел ли?
2. **Прочитай `index.md`** — все ли ссылки соответствуют реальным файлам?
3. **Просмотри `wiki/errors/`** — есть ли записи об ошибках, которые уже исправлены? Поставь пометку `> ✅ ИСПРАВЛЕНО (дата)` рядом
4. **Просмотри `wiki/skills/`** — есть ли навыки, которые устарели или противоречат новым? Добавь пометку `> ⚠️ УСТАРЕЛО (дата): причина`
5. **Проверь `raw/`** — есть ли логи сессий, где агент НЕ оставил «Что НЕ получилось»? Если да — добавь комментарий в `supervisor_report.md`

### Шаг 4: Обнови отчёт супервизора

После всех исправлений обнови `supervisor_report.md`:

```markdown
# Supervisor Report — [дата время]

## Выполнено
- [что исправлено]

## Требует внимания (не смог исправить автоматически)
- [описание проблемы]

## Рекомендации агентам
- [что агенты должны учесть в следующих сессиях]
```

### Шаг 5: Закоммить

```bash
cd /home/segun/CascadeProjects/Перед\ 0_2/AGENTS_RULES-v2
git add -A
git commit -m "[Jules-Supervisor] health check + cleanup"
git push origin main
```

## Критерии принятия решений

| Ситуация | Действие |
|----------|----------|
| Ошибка в wiki/errors/ уже исправлена в коде | Добавь `> ✅ ИСПРАВЛЕНО` + дата |
| Навык в wiki/skills/ противоречит новому | Добавь `> ⚠️ УСТАРЕЛО` + причина |
| Страница wiki > 15KB | Разбей на две тематические страницы |
| shared_status.md > 2KB | Сократи, оставь только актуальное |
| Лог сессии без «Что НЕ получилось» | Запиши в отчёт как нарушение протокола |
| Дубликаты записей | Удали, оставь только последнюю |
| Агент не делал Check-out > 3 дня | Запиши в отчёт как проблему координации |
