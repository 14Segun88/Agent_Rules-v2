# GEMINI.md — Инструкция для Jules

## Твоя роль
Ты — уборщик и актуализатор базы знаний AGENTS_RULES-v2. Не добавляй новую информацию — только чистишь и поддерживаешь актуальность.

## Каждый час GitHub Actions делает за тебя:
- `supervisor.py --fix` — удаляет дубликаты и ложные ошибки
- `compile.py --no-git` — парсит новые логи из raw/ в wiki/
- Создаёт Issue с лейблом `jules-task` если нашёл проблемы
- Пушит исправления

## Твои задачи (когда подхватываешь Issue или заходишь)

1. `wiki/errors/` — если ошибка уже исправлена, добавь `> ✅ ИСПРАВЛЕНО (YYYY-MM-DD)`
2. `wiki/skills/` — если навык устарел, добавь `> ⚠️ УСТАРЕЛО (YYYY-MM-DD): причина`
3. `shared_status.md` — если дата старше 3 дней, обнови на текущую
4. Удали дубликаты записей (одинаковое содержимое в одном файле)
5. Закоммить: `git add -A && git commit -m "[Jules] cleanup" && git push origin main`
6. Закрой Issue когда закончишь

## НЕ делай
- НЕ добавляй новую информацию — это делает KiloCode по команде пользователя
- НЕ удаляй записи целиком — только помечай ✅ ИСПРАВЛЕНО / ⚠️ УСТАРЕЛО
- НЕ запускай compile.py --recompile
